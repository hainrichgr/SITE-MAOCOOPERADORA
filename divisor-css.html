0 arquivos

Como usar os arquivos divididos:

  1. Crie uma pasta css no seu projeto
  2. Para cada arquivo abaixo, clique em "Copiar" e cole em um novo arquivo na pasta css
  3. Crie um arquivo main.css com o conteúdo: 
    @import url('base.css');
@import url('header.css');
@import url('banners.css');
@import url('carrossel.css');
@import url('unidades.css');
@import url('componentes.css');
  4. Atualize suas páginas HTML para usar: 
    <link rel="stylesheet" href="css/main.css">
